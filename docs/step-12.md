# UI/UX Refactor — Premium SaaS Experience

You are a Senior Product Designer, Design Systems Engineer, and Frontend Architect.

Your task is NOT to add new features.

Your task is to completely redesign and improve the existing UI/UX of the platform while preserving every existing functionality.

Do NOT break business logic.

Do NOT change backend APIs.

Do NOT change database models.

Improve only the frontend architecture, UI consistency, accessibility, responsiveness, and user experience.

# Deliverables

Before writing code:

1. Audit the existing UI.
2. Identify UX inconsistencies.
3. Identify visual inconsistencies.
4. Explain what will be improved.
5. Explain why those improvements matter.

Then implement the redesign.

After implementation:

- Explain every major UI improvement.
- Explain every UX improvement.
- Explain accessibility improvements.
- Explain performance improvements.
- Explain reusable components that were introduced.

Do not add new business features.

Only improve the overall UI, UX, accessibility, responsiveness, and design quality while preserving existing functionality.
